# SQL92-keywords

> is the list of operators in SQL92 specification

[Installation](#installation) |
[Usage](#usage) |
[License](#license)

## Installation

With [npm](https://npmjs.org/) do

```bash
npm install sql92-operators
```

## Usage

Get operators list.

```javascript
const operators = require('sql92-operators')

operators.indexOf('+') > -1 // true
```
